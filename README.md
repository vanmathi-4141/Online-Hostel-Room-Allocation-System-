**Online Hostel Room Allocation System**

A web-based application designed to simplify and automate hostel room allotment for students. This system streamlines the traditional manual process by providing an efficient, transparent, and user-friendly platform for managing applications, verifying student details, allocating rooms, and generating repository

**Installation & Setup**

1. Clone the Repository
git clone https://github.com/vanmathi-4141/Online-Hostel-Room-Allocation-System-.git

2. Start Local Server
Use XAMPP/WAMP if using PHP, or the appropriate backend server.

3. Import Database
Open phpMyAdmin

Create database:
hostel_db

Import the SQL file from /database

4. Run the Project
Open browser and go to:

http://localhost/Online-Hostel-Room-Allocation-System-
